== Description ==
This plugin generates dummy text using http://loripsum.net/api.

== Installation ==
1. Upload the `loreme-dummy-text-generator` plugin folder to the `/wp-content/plugins/` directory or install it using Plugins in WordPress.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use the shortcode [loremeX] to generate dummy text, X is the number of paragraphs.

== Changelog ==
= 1.2 =
* Fixed rounding issue with 0.5 numbers.

= How do I use the plugin? =
You can use the plugin by putting the shortcode [loremeX] in your editor remember  X is the number of paragraphs or sentences.
